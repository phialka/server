# About the project
The main idea of the project is to create a tool with which any confident PC user could deploy their own messaging server.
We plan to gradually implement this idea, which includes several stages:
1. Creating a working version of the messaging server.
2. Creating a minimal client.
3. Work on ways to easily deploy the server.
4. Adding additional functionality, correcting errors (a stage with no end)

If you want to take a closer look at this project and its implementation, we have prepared [documentation](docs/basic_info.md) for you.
